# python-web-dash-mvc-sqlserver-simple

## Description
Creates a small database table `dog`
for dash project.

## Tech stack
- python
  - dash
  - sqlalchemy
  - pandas

## Docker stack
- python:latest
- mcr.microsoft.com/mssql/server:2017-CU17-ubuntu

## To run
`sudo ./install.sh -u`
- [Avaible at](http://localhost)

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
